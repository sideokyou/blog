Kpopticket.us
The client requested me to build a website for selling concert tickets. Since Kpop popularity has been increasing, the client needed robust, fast, and clean website for their busienss.

Requirements

  • Must be secured ticket slot when a customer buys ticket(s)
  • Not allowed having duplicate purchased seats
  • Must have administration panel so that ticket management can be controlled by a client
  • Release ticket sales or event on set time
  • Must used 3rd party payment system which is Authorize.net for payment
  • Server must send online email which is composed by PDF to the purchaser
  • Each customer can check their order stauts or ticket info on My account page

Results

  • Used SVG for drawing seating chart so that it is so clear and easy to nagivate
  • When a customer select some seats, then database locks the seats so that no one can buy or select those seats
  • Developed a daily sale report to the client using cron job for complimentary
  • Modern and clean design for increasing user experience with mobile resposiveness

Stacks

  • Frontend - HTML, CSS, Bootstrap, jQuery, and Vanilla JavaScript
  • Backend - PHP and MySQL
  • 3rd party services - Authorize.net, Sendgrid
